# CareerOS AI Suite — Public Demo

An interactive SaaS dashboard showcasing AI‑driven career management and job‑matching analytics.

## 🚀 Overview
CareerOS AI Suite is a professional‑grade demo built to demonstrate full‑stack architecture, clean design, and applied AI workflows.  
It features a modern UI (React + Tailwind + shadcn/ui) and a lightweight FastAPI backend with Gemini embeddings.

## 🧠 Features
- Job pipeline management (Kanban)
- Offer CRUD
- AI match analysis (CV ↔ job)
- Dashboard metrics and reminders
- Dark/Light mode
- Demo data (Stripe, Figma, Vercel, Spotify)

## 🧩 Tech Stack
| Layer | Technology |
|-------|-------------|
| Frontend | React, TypeScript, Tailwind, shadcn/ui, Framer Motion |
| Backend | FastAPI, Pydantic |
| Database | Supabase (PostgreSQL + pgvector) |
| AI | Gemini 1.5 Flash, Hugging Face Transformers |
| Deployment | Vercel + Railway |

## 📈 Architecture
Clean monolithic architecture with modular services for scalability.  
Includes RAG pipeline and fallback logic for free AI models.
